# Learning

Use this folder for review artifacts generated from grounded wiki pages and recent reading.

- `flashcards/` stores spaced-repetition cards.
- `quizzes/` stores short retrieval-practice quizzes.
- `review_schedule.yaml` tracks scheduled reviews.

Every learning artifact should link to a source wiki page or raw source.

## Workflow

Use `learning-review` when recent reading or weak concepts should become retrieval practice. Prefer cards and quizzes that test explanation, comparison, and application.

## Flashcard Shape

```markdown
# Flashcards: Topic

## Card 1
Q: Question?
A: Concise answer.

Source: [[Source Page]]
Difficulty: 2
Review due: YYYY-MM-DD
```

## Quiz Shape

```markdown
# Quiz: Topic

## Q1
Question text.

Correct: Answer
Explanation: Brief explanation.
Source: [[Source Page]]
```

Do not create review artifacts from unsupported claims.
