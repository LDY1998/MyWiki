## Notes

| **Primitive**     | **What happens**                                                                                                                                            | **Typical use**                                          |
| ----------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------- | -------------------------------------------------------- |
| **AllReduce**     | Every GPU starts with its own tensor, all tensors are reduced together with an op like sum/max, and the final reduced result is copied back to **all** GPUs | Gradient synchronization in data parallel training       |
| **Broadcast**     | One source GPU sends the same tensor to all other GPUs                                                                                                      | Send model weights or config from one rank to everyone   |
| **Reduce**        | All GPUs send tensors to one destination GPU, where they are reduced into a single result                                                                   | Aggregate results to a master rank                       |
| **AllGather**     | Every GPU contributes its local tensor, and all GPUs receive the concatenation of everyone’s tensors                                                        | Gather sharded activations, embeddings, or parameters    |
| **ReduceScatter** | First reduce tensors across GPUs, then scatter chunks of the reduced result so each GPU gets only its own part                                              | ZeRO, sharded optimizers, distributed gradients          |
| **AlltoAll**      | Every GPU sends a different chunk to every other GPU, and receives one chunk from each                                                                      | Mixture-of-Experts token routing, some parallel shuffles |
| **Gather**        | All GPUs send tensors to one destination GPU, which collects them without reducing                                                                          | Collect outputs or diagnostics on one rank               |
| **Scatter**       | One source GPU splits a tensor into chunks and sends one chunk to each GPU                                                                                  | Distribute data or parameter shards from one rank        |
